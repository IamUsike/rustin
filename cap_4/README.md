#### **CAPSTONE PROJECT: Text Analyzer**

Build a command-line text analyzer that demonstrates ownership mastery. Read a multi-line string (hardcoded), then compute and print stats.

**Requirements:**

- Function to count words, sentences (by '.'), and unique characters
- Function to find the longest word (return a &str slice)
- Function to replace a word with another (return a new owned String)

---

**Additional Instructions:**

- All functions must use appropriate ownership/borrow semantics (no unnecessary clones)
- Print a formatted report with all statistics
