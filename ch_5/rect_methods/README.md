## Rectangle methods (easy)

Define a Rectangle struct with width and height.  
Add methods: `area()`, `perimeter()`, `is_square()`, and `can_hold(&self, other: &Rectangle) -> bool`.

Write a `main` that exercises all of them.

**Hint:**

```rust
impl Rectangle {
    fn area(&self) -> f64 {
        self.width * self.height
    }
}
```
