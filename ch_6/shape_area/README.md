**Shape area with enums**

Define a Shape enum with Circle(f64), Rectangle(f64, f64), Triangle(f64, f64, f64). Write fn area(shape: &Shape) -> f64 using match. Print areas for one of each shape.

_Hint: For Triangle use Heron's formula: s = (a + b + c) / 2, area = sqrt(s _ (s - a) _ (s - b) _ (s - c)).\*
